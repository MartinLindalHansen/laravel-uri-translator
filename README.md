Translate a URI or individual slugs, excluding route parameters.

## ✅ Requirements

- PHP >= 7.2.5
- Laravel >= 7.0

## 📦 Install

Install this package with Composer:

```bash
composer require codezero/laravel-uri-translator
```

Laravel will automatically register the ServiceProvider.
